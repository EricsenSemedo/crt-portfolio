# LEARNING.md

Owner knowledge profile for calibrating agent explanations and code comments.

## How This Works

This file tracks what you know, what you are learning, and what is new to you.
AI agents read this file to calibrate how much they explain. You do not need
to edit this file manually — agents update it through the `/learn` skill.

## Proficiency Levels

| Level | Meaning | Agent Behavior |
|-------|---------|----------------|
| `known` | You are fluent | Skip explanations |
| `familiar` | You understand but may be rusty | Brief reminders OK |
| `learning` | Actively building understanding | Explain with context |
| `new` | Just introduced | Full explanation with examples and WHY |
| `stale` | Known but not seen in 30+ days | Quick refresher |
